# 网易云音乐下载工具
#### by 陈暮尘
- 小工具： 用于下载网易云音乐VIP音乐（mp3 格式）
- 注意：无法下载试听曲库及付费专辑 
- 需要下载安装以下第三方库：
    - eyed3
    - pyqt5（运行图形化界面）
    - tqdm
- 运行以下命令启动：
    - 启动命令行 `python downloadNeteaseMusic.py`
    - 启动UI界面 `python Main.py`
